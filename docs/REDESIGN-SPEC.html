3. Graduation Analysis modal (opens when the banner is clicked)

A large modal with a left rail (current grad rate, edition/author meta, the 8 equipped gear pieces) and a right area with a horizontal tab bar. Tabs (keep all, in this order):

TabPurpose / content
Manual SheetEditable panel inputs grouped Physical / Attribute (Bamboocut) attributes; "How to use" help; Dungeon Target = Tier 91 / Lv95 (Global).
Stat PriorityTwo ranked lists (gain / loss): per stat, the grad-% change from adding/removing one typical substat roll. Live vs the current panel.
Cultivate (beta)Three sections: substat-count (条) progress tiles vs graduated targets; tuned-substat upgrade ranking; greedy next-8 substat advice.
ComparePer-slot: compare each gear piece, show which raises graduation most.
Transmute AdvicePick a slot and a substat to re-roll, then shows which replacement substat (at max T91 roll) yields the best graduation gain + a verdict.
Best BuildScans the whole gear pool, finds the highest-graduation 8-piece combination; shows top combo + alternatives + "Equip this build".
(A Transmute Cooldown tab — 7-day CD countdown per transmuted item — exists in the CN reference and may be added later; leave room for one more tab.)
