# CodeWhale for Antigravity

CodeWhale is an autonomous AI coding agent running DeepSeek V4 Pro.

## Quick Start

After installation, restart Antigravity. The agent will auto-detect the skill and MCP server.

## Usage Examples

- "CodeWhale, debug why the login API returns 500"
- "Use CodeWhale to explore the codebase and find all deprecated API calls"
- "CodeWhale, refactor the user service to use the new database schema"
- "Spawn 3 CodeWhale sub-agents to check auth, database, and frontend in parallel"

## Requirements

- CodeWhale CLI installed (`npm i -g codewhale` or equivalent)
- CODEWHALE_API_KEY environment variable set
